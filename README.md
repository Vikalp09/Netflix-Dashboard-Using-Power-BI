# Netflix-Dashboard-Using-Power-BI
This project is an interactive Netflix Dashboard built using Power BI to analyze and visualize Netflix content data. The dashboard provides insights into movies and web series based on IMDb ratings, awards, release trends, runtime distribution, and global availability.
